# I2C Scanner for CircuitPython

This demo scans the I2C bus and prints the addresses of any devices found to the serial console.

## Usage
1. Connect your I2C devices to the SCL and SDA pins.
2. Power on your board.
3. Open a serial console (e.g., `tio /dev/ttyACM0` or the Mu editor) to see the output.

## Hardware Setup (Raspberry Pi Pico)
- **SCL**: GP1
- **SDA**: GP0
- **VCC**: 3.3V
- **GND**: GND
